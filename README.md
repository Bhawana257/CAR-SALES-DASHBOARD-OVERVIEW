Car Sales Performance & Analytics Dashboard
📊 Project Overview
This project features a high-impact Car Sales Dashboard designed to track and analyze key performance indicators (KPIs) for a dynamic automotive retail environment. Using a combination of Power BI and Excel, this dashboard provides deep insights into sales trends, inventory movement, and regional performance, allowing stakeholders to make data-driven decisions regarding pricing and stock.

🛠️ Tech Stack & Skills
Data Visualization: Power BI (Desktop & Service)

Data Processing: Power Query for ETL (Extract, Transform, Load)

Calculations: Advanced DAX (Data Analysis Expressions) for YTD, MTD, and Year-over-Year (YoY) growth metrics.

Statistical Analysis: Python/Jupyter Notebook (integrated for advanced data cleaning/EDA).

Formatting: Custom UI/UX design with dark mode for enhanced readability.

📈 Key Metrics & Insights
The dashboard is split into Overview and Details views to provide both high-level summaries and granular data:

Financial KPIs:

YTD Total Sales: $371.2M (with MTD tracking at $54.28M).

YTD Average Price: $27.99K per vehicle.

YTD Cars Sold: 13.26K units.

Sales Trends: A weekly trend analysis highlighting peak sales periods and seasonality.

Customer Preferences:

Body Style: Breakdown of sales across SUVs, Hatchbacks, Sedans, and Hardtops.

Color Analysis: Pale White, Black, and Red identified as leading consumer choices.

Regional & Dealer Performance: * Geospatial mapping of sales by Dealer Region across the US (Austin, Janesville, Scottsdale, etc.).

Company-wise Sales: Competitive analysis ranking brands like Chevrolet, Ford, and Dodge by total revenue and market share.

📂 Features
Dynamic Filtering: Users can filter the entire dataset by Body Style, Dealer Name, Transmission, and Engine Type.

Detailed Records: A dedicated "Details" tab providing a raw look at individual transactions, including Customer Name, Dealer Name, and specific car models.

Interactive UI: Navigation sidebar to switch seamlessly between overview metrics and deep-dive tables.
